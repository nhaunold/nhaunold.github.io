Xpandr is a shortcut expander for the iPhone / iPod touch. It allows you to define an unlimited number of shortcuts to save time and make your life easier.

It works in every application, even on AppStore applications. It supports simple textfields like Spotlight and multiline textfields like Notes.app, Beejive or Mail.
  • If you're constantly making the same typo, Xpandr will fix this for you.
  • If you need a default reply for mails, Xpandr can do this as well.
  • Or if you just don't want to enter the same text over and over again, Xpandr will help you here too.
