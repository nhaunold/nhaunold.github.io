Help & Examples

What is Xpandr?

Xpandr adds powerful features to the integrated iOS keyboard shortcut system on your iPhone/iPod touch/iPad. For example, Xpandr allows you to expand shortcuts immediately as you type them. Don't worry, that's not all. You can check out a full feature list below.

In short: Xpandr makes you more productive & helps you save time while doing so. Awesome, isn't it?

Expand Immediately

You've been there. Just as you're about to send off that important email, you accidentally dismiss the shortcut suggestion bubble. With Xpandr, all these little troubles are gone. Expand Immediately expands your shortcuts as you type them, in real-time. You never have to worry about missing a shortcut ever again.

Stay Up-to-Date

Xpandr also adds more features to the shortcut system itself. Ever wanted to include the current date and time in your shortcut? Xpandr now allows you to do exactly that. For Geeks: It's fully compatible with RFC3339.

Check out some examples below!

Save time

The new version of Xpandr now keeps track how many times you use it and generates accurate statistics for your viewing pleasure. Keep track of how many hours you have saved thanks to Xpandr.

Automate

Xpandr can now automatically execute scripts whenever you type a shortcut. Ever wanted to have a shortcut which prints how much diskspace you have left on your device? With Xpandr, it's now possible.

Take a look at some examples below.

Documentation

Date Syntax

You can include RFC3339-compatible dates in your shortcuts by using the date command. For example: {date:yyyy-MM-dd HH:mm:ss} will expand to 2012-08-10 01:54:23.

Scripting Syntax

Execute scripts and programs by using the cmd command. For eaxmple: {cmd:/var/mobile/i_am_awesome.sh} will execute the bash script at /var/mobile/i_am_awesome.sh and will insert everything it prints to stdout.

Examples

Date: Fr., Aug 10, '12

{date:EEE, MMM d, ''yy}

Date: 2012-08-10 at 02:04:12

{date:yyyy-MM-dd 'at' HH:mm:ss}

Scripting: Device Uptime

Shortcut: {cmd:/var/mobile/uptime.sh}

/var/mobile/uptime.sh:
#!/bin/bash
echo -n `uptime`
